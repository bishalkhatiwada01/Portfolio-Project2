Development of the Site

06/04/2022

Making decisions for User interface of the site was challenging. i researched many portfolio websites in the internet. It will be true to say that i was inspired by such designs but i decided to keep it simple in terms of user interface design. there are many components of user interface that would make difference. components such as Font family, style, sizes and even font weight. When and where to use. besides font there are major components such as background colours etc.

I decided to go with Poppins font family because Poppins is one of the Geometric sans serif typefaces have been a popular design tool for building websites. Each letterform is nearly monolinear, with optical corrections applied to stroke joints where necessary to maintain an even typographic color. this font is great for both headlines and paragraph copy to improve readability and style. I used Google fonts to import and apply this font.

And as for the Colours, I used different colours as i was developing. coolors.co is one specific site that helped me for awesome colour palette. I used the color property in CSS to style the text colours whereas, background-color is used for formating background colour of the page, section or a div. Besides this I have also used icons and icons8 website was go to place for me as there are many icons. same icons with different size. so the variety offered in this website was very nice.
